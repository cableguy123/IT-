# 패스키 

> 비밀번호를 대체하는 더 안전하고 쉬운 방식 
- 생체 인식 센서
- PIN 패턴
- FIDO Aliance,W3C(World Wide Web Consortium)의 표준인 WebAuthn(웹 인증)
- 즉,생체 인식 정보나 PIN번호로 본인 인증을 수행 

- Android 4.4(API Level 19)이상에서만 지원
