# TLS,SSL 

- TLS (Transport Layer Security) 과거 명칭은 SSL(Secure Sockets Layer)
- TCP/IP 네트워크를 사용하는 통신에 적용
- TLS는 클라이언트/서버 응용 프로그램이 네트워크로 통신을 하는 과정에서 도청,간섭,위조 방지위해 설계 

## 절차
1. 지원 가능한 알고리즘 서로 교환
2. 키 교환,인증
3. 대칭키 암호로 암호화하고 메세지 인증 
---
## 네트워크 보안 구성 
- 인증 기관(CA) 맞춤설정 
- 디버그 전용 재정의
- 일반 텍스트 트래픽 선택 헤제
- 인증서 고정
